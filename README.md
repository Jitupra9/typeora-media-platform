
# Typeora

Typeora is a modern platform built using the **MERN stack** (MongoDB, Express, React, Node.js) that provides users with a unique experience of **reading news articles** in a concise format. It offers a variety of topics, trending news, and live streaming options. The platform also supports users from different organizations, allowing them to **go live**, broadcast events, and share content. It’s the perfect place for users who want to stay informed with bite-sized news updates and interact with live content.

## Features

* **News Articles Summarized**: Get quick summaries of news articles in multiple categories.
* **Explore More**: Users can explore full articles if they want to read more details.
* **Trending News**: Stay updated with the most trending news in real-time.
* **Live Streaming**: Watch live videos and broadcasts on a variety of topics.
* **Multiple User Roles**:

  * **Normal Users** can read articles, watch live streams, and explore content.
  * **Organizational Users** can go live and stream content to the community.
* **Real-time Updates**: Live feeds and real-time news updates for an engaging experience.

## Technologies Used

* **MongoDB**: NoSQL database to store user data, articles, and live stream content.
* **Express**: Web framework for Node.js to handle routing and server-side logic.
* **React**: Frontend JavaScript library to build a responsive and dynamic user interface.
* **Node.js**: JavaScript runtime for building the server-side application.
